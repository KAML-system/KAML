⚠️ مشكلة في صلاحيات الوصول

يبدو أن هناك مشكلة في قواعد الأمان في Firestore. لحل هذه المشكلة:

  1. اذهب إلى Firebase Console → Firestore Database → Rules
  2. استبدل القواعد الحالية بالتالي (للاختبار فقط):
    3. rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
  3. انقر على Publish لحفظ التغييرات
  4. انتظر دقيقة واحدة ثم اضغط تحديث البيانات

تنبيه: هذه القواعد للاختبار فقط. يجب تحديثها لاحقاً لتكون أكثر أماناً.
